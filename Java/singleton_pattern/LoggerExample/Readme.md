# Singleton Pattern Example: Logger

This example demonstrates the Singleton design pattern in Java using a simple Logger class.

## Files
- `Logger.java`: Singleton Logger class
- `Main.java`: Demonstrates usage and verifies singleton property

## How to Run

1. Compile:
   ```sh
   javac Logger.java Main.java
   ```
2. Run:
   ```sh
   java Main
   ```

You should see output indicating both logger instances are the same, and log messages printed to the console.
